What's a playable story?

The AI turns your material into a branching narrative. The learner reads a scene, makes a choice, and the story responds — tracking variables, revealing consequences, reaching different endings.

You're the new project manager. The deadline is in two weeks, and the team just lost a key developer. → Push the team to hit the deadline → Negotiate for more time → Reassign priorities quietly Each choice leads to a different path. Variables like trust and experience shift based on decisions.

The result is a self-contained HTML file — works offline, share via email or LMS, no install needed.
